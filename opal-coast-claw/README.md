# Opal Coast Claw Website

## 🎨 Nail Salon Website - BIAB Specialist

A beautiful, modern, and aesthetic single-page website for Opal Coast Claw, a boutique nail studio specializing in BIAB (Builder In A Bottle) nails in Sentul City, Bogor.

## ✨ Features

### Design
- **Aesthetic**: Elegant, feminine, luxurious but minimal
- **Color Palette**: Pearl white, dusty rose, iridescent lavender, gold accents
- **Typography**: Cormorant Garamond (serif) + Inter (sans-serif)
- **Animations**: Smooth scroll animations with AOS library
- **Responsive**: Mobile-first design

### Sections
1. **Hero**: Full-screen gradient with CTA to WhatsApp
2. **About**: Boutique studio intro with features
3. **Services**: BIAB focus with service cards
4. **Gallery**: Masonry photo grid
5. **Reviews**: Google 5.0 rating with testimonials
6. **Contact**: WhatsApp CTA + Google Maps
7. **Footer**: Links and social media

### Technical
- Pure HTML/CSS/JS (no backend needed)
- Tailwind CSS for styling
- AOS library for scroll animations
- Lucide icons
- Google Fonts
- WhatsApp floating button (always visible)
- Fast loading, optimized

## 🚀 Usage

### Option 1: Direct Open
Simply open `index.html` in any modern browser.

### Option 2: Local Server (for testing)
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Using PHP
php -S localhost:8000
```

Then open: http://localhost:8000
